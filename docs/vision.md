# となりっこ — プロダクトビジョン

**となりっこ — うちの子を覚える育児AI**（うちの子のこと、覚えながら聞く）

## 誰のため

- **0〜6歳**のお子さんを育てる保護者（7歳以上も利用可・注意表示あり）
- 日々の育児の不安を、短く・落ち着いて相談したい人

## 提供価値

1. **パーソナライズ** … 会話からうちの子のことを覚え、相談が続くほど寄り添いが深まる
2. **週次振り返り（Plus）** … 先週の育児とあなた自身のことを、自分を責めずに振り返れるレポート（お子さん・保護者それぞれ）
3. **複数の子・関係（Plus）** … きょうだいや友達の関係を踏まえた相談（任意）
4. **あなた（保護者）の相談（Plus）** … 疲れや気持ちに、登録済みのお子さんの情報を踏まえて寄り添う

## プラン方針

| 期間 | 内容 |
|------|------|
| 初回14日 | Plusと同じ機能（無制限・覚えながら更新・複数子・関係・あなた（保護者）の相談・週次） |
| 以降 Free | 1日5回・これまで覚えた分を相談に活用（新規更新なし） |
| Plus | すべて解放・月額課金 |
