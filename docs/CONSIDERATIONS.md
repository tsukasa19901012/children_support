# 仕様・考慮事項

## プロダクトポジション

| 項目 | 内容 |
|------|------|
| サービス名 | **となりっこ** |
| ターゲット | **0〜6歳**のお子さんを育てる保護者（7歳以上も可） |
| 提供価値 | うちの子を覚える育児AI（メモリ）＋週次振り返り（Plus）＋保護者本人の相談（Plus） |
| 競合との差 | 汎用24h相談ではなく、**継続メモリ＋週次レポート** |

表示文言は `src/lib/brand.ts` を正とする。

## 対象年齢

| 項目 | 内容 |
|------|------|
| 推奨 | **0〜6歳** |
| 利用 | **7歳以上**も可能（注意表示のみ） |
| AI | 0〜6歳向けに最適化。年齢外は可能な範囲で支援 |

## プラン・課金

| 項目 | 内容 |
|------|------|
| プラン | **Free** / **Plus** のみ |
| 初回14日 | Plus相当（案Cトライアル） |
| Free（トライアル後） | 1日5回・メモリ**読み取りのみ**・子1人 |
| Plus | 無制限・メモリ更新・複数子・関係・保護者相談・週次 |
| 価格 | Plus **¥980/月** — Stripe `STRIPE_PRICE_ID_PLUS` |
| `plan` 列 | `free` / `plus` のみ。Webhook / service_role のみ更新 |

## 子ども・関係性（Plus / トライアル）

| 項目 | 内容 |
|------|------|
| 複数登録 | Plus またはトライアル中（DB トリガー） |
| 関係 | 任意。きょうだい・友達など |
| Plus→Free 後 | データは残る。相談は1プロフィールを選択。メモリ更新停止 |

## 保護者プロフィール（Plus / トライアル）

| 項目 | 内容 |
|------|------|
| 概要 | 保護者本人の疲れ・気持ちを相談するプロフィール（`profile_type = caregiver`） |
| 登録数 | **1人まで** |
| 登録項目 | 名前（必須）、誕生日・性別（任意） |
| お子さんとの関係 | 手動で「誰の保護者か」を複数選択（`guardian` 関係） |
| AI | 登録済みお子さんの年齢・記録を読み取り、家庭状況に合わせて回答 |
| 削除 | お子さんとは独立して削除可。**お子さん（child）は最低1人必須** |
| Free | 利用不可（Plus またはトライアル中のみ） |

## チャット

| 項目 | 内容 |
|------|------|
| メモリ読み取り | 全プラン |
| メモリ更新 | Plus・トライアルのみ |
| Free 制限 | 送信前に user メッセージを DB 挿入してから件数カウント |
| 履歴日数（UI） | Free: 14日、Plus/トライアル: 無制限 |
| AI へ送る直近件数 | Free: 5件、Plus/トライアル: 30件 |
| 週次レポート | Plus 契約中 **または** 14日トライアル中。**登録済みプロフィールごと**（お子さん・保護者）に送信（月曜 8:00 JST） |

## マイページ UX

| 項目 | 内容 |
|------|------|
| 編集から戻る | `router.back()` + 必要時のみ子どもデータをクライアント再取得 |
| 相談切替 | カード右上（相談中ラベルと同位置） |
