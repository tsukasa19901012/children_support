# 仕様・考慮事項

## プロダクトポジション

| 項目 | 内容 |
|------|------|
| サービス名 | **となりっこ** |
| ターゲット | **0〜6歳**のお子さんを育てる保護者（7歳以上も可） |
| 提供価値 | うちの子を覚える育児AI ＋ 週次振り返り（Plus）＋ あなた（保護者）の相談（Plus） |
| 競合との差 | 汎用24h相談ではなく、**うちの子を覚えながらの相談＋週次レポート** |

表示文言は `src/lib/brand.ts` と `src/features/billing/plans.ts` を正とする。

## 対象年齢

| 項目 | 内容 |
|------|------|
| 推奨 | **0〜6歳** |
| 利用 | **7歳以上**も可能（注意表示のみ） |
| AI | 0〜6歳向けに最適化。年齢外は可能な範囲で支援 |

## プラン・課金

| 項目 | 内容 |
|------|------|
| プラン | **Free** / **Plus** のみ |
| 初回14日 | Plusと同じ機能（14日体験） |
| Free（体験期間後） | 1日5回・子1人 |
| Plus | 無制限・覚えながら更新・複数子・関係・あなた（保護者）の相談・週次 |
| 価格 | Plus **¥980/月** — Stripe `STRIPE_PRICE_ID_PLUS` |
| `plan` 列 | `free` / `plus` のみ。Webhook / service_role のみ更新 |

## 子ども・関係性（Plus / 体験期間）

| 項目 | 内容 |
|------|------|
| 複数登録 | Plus または体験期間中（DB トリガー） |
| 関係 | 任意。きょうだい・友達など |
| Plus→Free 後 | データは残る。相談は1人分を選択。会話からの記憶更新は停止 |

## あなた（保護者） / caregiver（Plus / 体験期間）

| 項目 | 内容 |
|------|------|
| 概要 | あなた（保護者）として、疲れや気持ちを相談する（`profile_type = caregiver`） |
| 登録数 | **1人まで** |
| 登録項目 | 名前（必須）、誕生日・性別（任意） |
| お子さんとの関係 | 手動で「誰の保護者か」を複数選択（`guardian` 関係） |
| AI | 登録済みお子さんの情報を踏まえ、家庭状況に合わせて回答 |
| 削除 | お子さんとは独立して削除可。**お子さん（child）は最低1人必須** |
| Free | 利用不可（Plus または体験期間中のみ） |

## チャット

| 項目 | 内容 |
|------|------|
| 記憶の反映 | 全プラン（`children.memory` をプロンプトに載せる） |
| 記憶の更新 | Plus・体験期間中のみ（会話後に非同期更新） |
| Free 制限 | 送信前に user メッセージを DB 挿入してから件数カウント |
| 履歴日数（UI） | Free: 14日、Plus/体験期間: 無制限 |
| AI へ送る直近件数 | Free: 5件、Plus/体験期間: 30件 |
| 週次レポート | Plus 契約中 **または** 14日体験期間中に配信。**お子さん・あなた（保護者）** ごと（月曜 8:00 JST）。Free は過去分の閲覧可 |

## マイページ UX
